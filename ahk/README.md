## 快捷键选择标准

为了减少过多的快捷键导致的记忆负担，我们为快捷键的设计确立了一些标准：

- 全局快捷键：`Space+功能名首字母` 
- 特定软件快捷键：`Alt+功能名首字母`
- 某功能的相关（类似）功能的快捷键应与原快捷键类似
	- 将 `Alt+xx` 改为`Shift+Alt+xx`  
	- 将 `Space+xx` 改成 `F4+x`
	- 其他类似的快捷键
