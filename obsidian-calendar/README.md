# obsidian-calendar

obsidian日历插件，支持循环事件

## 使用步骤

- 安装 `Obsidian`， `Dataview` 插件 和 `Templater` 插件
- 创建一个文件夹用来存放日程
- 在`Templater` 插件的设置中，启用 `loop.md` 作为该文件夹的 Folder Template，以创建循环事件
- 将 `front-matter-template.md` 放置在这个文件夹中
- 将 `calendar.js` 压缩后，包裹在 `dataviewjs` 代码块中，同样放置在这个文件夹中，便可以查看日程
